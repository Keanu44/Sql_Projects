# Exploratory Data Analysis using SQL: U.S. Baby Names Trends Analysis

- This SQL project explores historical baby name data to uncover cultural trends, regional differences, and changing popularity patterns over time using real-world demographic datasets.

## Executive Summary

**Project Scope:** Developed multiple analytical SQL queries to analyze name popularity by gender, decade, and region.

**Data Modeling:** Combined fact table (names) with regional dimension table (regions) using joins and data-cleaning CTEs.

**Outcomes:** Identified long-term naming trends, regional preferences, and the rise of gender-neutral names through time-series and aggregation analysis.

## Tech Stack

- Query Engine: DuckDB for fast in-memory analytics

- Language Used: SQL (CTEs, Window Functions, Aggregations)

- Development Environment: VS Code + DuckDB CLI

- Version Control: Git & GitHub for reproducible SQL workflows.

## Key Insights

- A small number of names dominate historically, but popularity cycles shift every generation.

- Cultural influence (media, celebrities, societal norms) strongly impacts naming spikes.

- Regional analysis reveals localized naming identities across the U.S.

- Gender-neutral naming is not new but is becoming more statistically significant.

- SQL alone can deliver deep sociological insights without external analytical tools.
