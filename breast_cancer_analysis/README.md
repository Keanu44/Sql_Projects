# Breast Cancer Survival Analysis (SQL | DuckDB | MotherDuck)

## 📌 Project Overview

**This project analyzes breast cancer patient data to investigate:**

- Survival outcomes

- Mortality risk factors

- Stage severity impact

- Tumor size influence


The objective is to simulate a real-world healthcare analytics workflow, starting from raw CSV ingestion to cleaned dataset creation and advanced survival analysis using SQL.


## 🔍 Key Analytical Questions

**How Does Cancer Stage Impact Survival ?**

**Does Tumor Size Increase Mortality Risk ?**

**Which Patients Outperform Stage Expectations ?**

**Are There Demographic Differences in Mortality ?**


## 🧠 Key Takeaways

**This project demonstrates:**

- End-to-end SQL workflow

- Data cleaning & transformation

- Risk segmentation

- Survival benchmarking

- Healthcare-style KPI analysis

- Analytical thinking beyond basic querying.

## 🧬 Key Insights from the Breast Cancer Analysis

- Advanced Cancer Stage Strongly Increases Mortality Risk

- Larger Tumor Size Correlates with Higher Mortality

- Survival Within the Same Stage Varies Significantly

- Mortality Distribution is Not Uniform Across Demographics
